# MirITeam project, Stage 2

Обнаружение пробок, нарушений пдд и многое другое

## Как начать

Для запуска программы необходим Python 3.7.3 и библиотеки, которые будут описаны ниже. Также будут необходимы ВСЕ файлы, которые идут с кодом.
Также будет приложен архив с .exe файлом, который можно просто запустить.

### Требования

Для работы программы необходимо установить:

```
pip install opencv-python
pip install opencv-contrib-python
pip install matplotlib
pip install numpy

```
Для запуска .exe файла действия, описанные выше, не нужны.

## Запуск тестов

Для запуска теста программы необходимо сначала выбрать видеофайл, затем выбрать одну из функций

## Было использовано в процессе разработки

* [OpenCV](https://opencv.org/) - Открытая библиотека компьютерного зрения
* [YOLO](https://pjreddie.com/darknet/yolo/) - Нейронная сеть для обнаружения объектов
